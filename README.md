# Français pour bébas

An accelerated French primer that leverages Spanish fluency via cognate-transformation rules. Interactive mobile web app with pre-generated ElevenLabs audio.

- **Trip target:** Nice, summer 2026
- **Bridge language:** Spanish → French
- **Audio:** ElevenLabs (voice: Emilie, Parisian)

Live: https://bmount.github.io/frances-para-bebas/
